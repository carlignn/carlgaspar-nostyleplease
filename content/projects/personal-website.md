My personal website and blog where I write about security, homelab, projects, and life. Built from a customized [no style, please!](https://github.com/riggraz/no-style-please) Hugo template with a minimal design philosophy — no styles, just content. Yes, this post is about the very website you're reading right now - [carlgaspar.com](https://carlgaspar.com/).

## Features

- Minimal, lightweight, and fast
- Light, dark, and auto theme support
- Blog posts organized by category: security, homelab, projects, and life
- Comment system powered by Giscus (GitHub Discussions)
- Full-text search
- CMS integration for content management

## Technologies Used

* Hugo
* TinaCMS
* Netlify
* Giscus
